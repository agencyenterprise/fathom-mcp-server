Fathom MCP Server

Connect Claude to your Fathom meetings, transcripts, and AI summaries.

Open Source MIT License MCP Compatible
Add to Claude Desktop
https://www.fathom-mcp-server.com/mcp

Available Tools

list_meetings
List meetings with filters (date, team, recorder, etc.)
search_meetings
Search meetings by title with optional filters
get_transcript
Get full transcript for a recording
get_summary
Get AI-generated summary for a recording
list_teams
List all accessible teams
list_team_members
List members of a team
